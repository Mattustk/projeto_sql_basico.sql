# PROJETO SQL BÁSICO

## SOBRE O PROJETO
Projeto simples de banco de dados demonstrando:
- Criação de tabelas
- Inserção de dados
- Consultas com JOIN

## ESTRUTURA DAS TABELAS
- clientes (id_cliente, nome, email, telefone, data_cadastro)
- produtos (id_produto, nome, preco, estoque)
- pedidos (id_pedido, data_pedido, valor_total, id_cliente)
- itens_pedido (id_item, quantidade, subtotal, id_pedido, id_produto)

## CONSULTAS REALIZADAS
- Cliente que mais gastou
- Produto mais vendido  
- Pedidos por valor total

## COMO EXECUTAR
1. Acesse https://sqliteonline.com
2. Cole o código do script.sql
3. Clique em Run
